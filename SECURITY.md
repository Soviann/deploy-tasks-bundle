# Security Policy

## Supported versions

| Version            | Supported          |
| ------------------ | ------------------ |
| 0.x (pre-release)  | :white_check_mark: |

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

- **Preferred:** report privately via [GitHub Security Advisories](https://github.com/Soviann/deploy-tasks-bundle/security/advisories/new). This keeps the discussion confidential until a fix is released.
- **Alternative:** email `vasse.nicolas@gmail.com` if you do not have a GitHub account.

Please include:

- Affected bundle version(s)
- A minimal reproduction or proof of concept
- The impact you observed
- Any mitigation you have already identified

## Disclosure policy

- Reports are acknowledged within 7 days.
- A coordinated release date is agreed with the reporter once a fix is ready.
- Public disclosure happens no later than 90 days after the initial report, whether or not a fix has shipped.
- Credit is given in the release notes unless the reporter asks to stay anonymous.
